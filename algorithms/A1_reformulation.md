# A1 — m = n+1 変形（Riesel 形式）

## 動機

第一種カニンガム鎖（長さ k、先頭 n = p）は、k 個の数

```
f_i(n) = 2^i · n + (2^i − 1),   i = 0,1,…,k−1
```

が全て素数であること（f_0 = n, f_1 = 2n+1, f_2 = 4n+3, …）。

ここで `f_i(n) + 1 = 2^i (n + 1)` と因数分解できる。よって `m = n + 1` と置くと

```
f_i(n) = m · 2^i − 1,   i = 0,1,…,k−1
```

となる。すなわち**第一種カニンガム鎖とは、`m·2^i − 1` が連続する i について
全て素数になるような m を探すこと**である（m = p+1）。

これは Riesel 型の素数列 `m·2^i − 1` を「連続 i で」探す問題に帰着する。
Sophie Germain 素数（k=2）や Cunningham 鎖の標準的記述そのもの。

## 悪残基が「2 の逆べき」になる

奇素数 q が `m·2^i − 1` を割るための条件は

```
m · 2^i ≡ 1 (mod q)  ⇔  m ≡ (2^i)^{-1} (mod q)
```

したがって、n ではなく **m について篩う**とき、各素数 q の「悪い残基」は

```
BadSet_q = { (2^i)^{-1} mod q : i = 0,1,…,k−1 }
```

というただの逆べきの集合になる。利点：

* 計算が非常に安い（`2^i mod q` を順次 2 で掛け、その逆元を取るだけ）。
* 集合の大きさは `min(k, ord_q(2))` を超えない。`ord_q(2)`（2 の mod q での
  位数）が k より小さいときは残基が重複し、より強く篩える。
* q = 2 のときは `m·2^i − 1 ≡ −1 ≡ 1 (mod 2)` で常に奇、悪残基なし
  （m は任意、ただし n = p が奇素数なので m は偶数に制限される）。

n 形式の悪残基 `r = −(2^i−1)·(2^i)^{-1} mod q` と、`m = n+1` として
`r+1 ≡ (2^i)^{-1} (mod q)` になることの一致は `A2_bad_residues.jl` で検証済。

## なぜ巨大数で効くか

* 鎖の各項の大きさは `m·2^i − 1`。i が大きい項は m より 2^i 倍大きいが、
  篩（小素数での割り算）は m についての剰余だけで決まり、各項の絶対値に
  依存しない。つまり**篩コストは m の桁数に依存せず、k と篩上限 P だけで決まる**。
* 最終判定（MR）だけが多倍長演算になる。したがって巨大数では「篩で
  候補を極限まで絞り、MR は本当に残った少数にだけ回す」構成が最適。
  これが A3/A4/A5 の設計原理。

## 例

m = 6 (p = 5) のとき：
`5 (=6·1−1), 11, 23, 47` は全て素数、次の `95` は合成 ⇒ CC4 の先頭は p=5。

m = 12 (p = 11)：`11, 23, 47` は素数、`95` 合成 ⇒ CC3 の先頭は p=11。
