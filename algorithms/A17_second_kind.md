# A17 — 第二種カニンガム鎖への適用（p, 2p−1, 4p−3, …）

第一種が `f_i(n) = 2^i n + (2^i − 1)`（＝`m·2^i − 1`, `m=n+1`）だったのに対し、
**第二種**は `g_i(n) = 2^i n − (2^i − 1)`。鎖は `p, 2p−1, 4p−3, 8p−7, …`。

`m' = n − 1` と置くと `g_i = m'·2^i + 1`。つまり第二種は「`m'·2^i + 1` が連続 i で
全て素数」になること。悪残基は第一種の `+(2^i)^{-1}` が `−(2^i)^{-1}` に変わるだけ。

```
BadSet_q^(2) = { −(2^i)^{-1} mod q : i = 0..k−1 }
```

したがって A2–A6 の全手法は「悪残基の符号を反転」だけで第二種に転用できる。
記録探索では第一種と第二種の両方が対象になるので、この 1 行の差を切り替えられる
ようにしておくと良い。実装は `A17_second_kind.jl`（自己検証付き）。
